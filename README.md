# 🛍️ Customer Shopping Behavior Analysis

## 📌 Overview

This project performs an end-to-end analysis of customer shopping behavior using Python, MySQL, and Power BI. The objective is to identify purchasing patterns, customer demographics, and business insights that help improve decision-making.

The project covers the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL querying, interactive dashboard creation, and business reporting.

---

## 📂 Dataset

The dataset contains customer shopping information, including:

- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount
- Location
- Size
- Color
- Season
- Review Rating
- Subscription Status
- Shipping Type
- Discount Applied
- Promo Code Used
- Previous Purchases
- Payment Method
- Frequency of Purchases

## 📋 Project Workflow

### 1. Data Loading
- Imported CSV dataset using Pandas.
- Explored dataset structure.
- Checked data types and summary statistics.

### 2. Data Cleaning
- Removed duplicates.
- Handled missing values.
- Converted data types.
- Created Age Groups.
- Prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)
- Customer demographics analysis.
- Purchase amount analysis.
- Product category analysis.
- Seasonal shopping trends.
- Subscription analysis.
- Review rating distribution.
- Payment method analysis.

### 4. SQL Analysis
Performed SQL queries to answer business questions such as:

- Total Revenue
- Average Purchase Amount
- Top Selling Categories
- Customer Distribution
- Seasonal Sales
- Subscription vs Non-Subscription Customers
- Highest Spending Customers
- Review Rating Analysis

### 5. Power BI Dashboard
Created an interactive dashboard featuring:

- Total Customers
- Total Revenue
- Average Purchase
- Sales by Category
- Revenue by Season
- Gender Distribution
- Age Group Analysis
- Subscription Status
- Payment Method
- Interactive Filters & Slicers

## 🚀 How to Run

### Python

1. Clone this repository.
2. Install required libraries:

```
pip install pandas numpy matplotlib seaborn
```

3. Open the Jupyter Notebook.
4. Run all cells.

### MySQL

1. Create a database.
2. Import the dataset.
3. Execute the SQL queries.

### Power BI

1. Open the `.pbix` file.
2. Refresh the dataset connection.
3. Explore the dashboard.
